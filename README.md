# Currency Converter API

## Summary

An API providing a range of currency conversion utilities.

Version: 5b8d0fd276b6d288905ed2f63a934e057e8feca2

## Installation

Clone this repository.

Run the following command

```
$ npm install
```

## Commands

To start the project:

```
$ npm start
```

To run tests:

```
$ npm test
```
