# Equal Experts Currency Converter Microservice

## Service Summary

A currency conversion utilities API Repo.

Version: 5b8d0fd276b6d288905ed2f63a934e057e8feca2

## Prerequisites

- [Node v16.13.2](https://nodejs.org/en/download/)

## Installation

Clone this repository.

Run the following command:

```bash
$ npm install
```

## Running the service

To start the project:

```bash
# production
$ npm run start

# development
$ npm run start:dev

```

## Running tests

To run tests:

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Read service docs

To read the open api docs for this service and try the api.

```bash
# start the service
npm start

# navigate to the following url in the browser: http://localhost:3001/docs
```
